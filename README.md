# 💬 WebSocket Chat App

A real-time chat application built using **Spring Boot**, **React**, and **WebSocket (STOMP + SockJS)**.

## 🚀 Features

-  Real-time messaging without page refresh
-  Multi-user chat support
-  Live connection status
-  Modern responsive UI
-  Fast frontend with React + Vite
-  WebSocket communication

---

##  Tech Stack

### Frontend
- React.js
- Vite
- CSS3
- SockJS Client
- STOMP.js

### Backend
- Spring Boot
- Spring WebSocket
- Maven
- Java

---

##  Project Structure

```plaintext
websocket-chat-app/
├── backend/
│   └── Demo_WebSocket
├── frontend/
│   └── React Chat App
```

⚙️ How to Run Locally
1️⃣ Backend
cd backend/Demo_WebSocket
mvn spring-boot:run

Runs on:

http://localhost:8080
2️⃣ Frontend
cd frontend
npm install
npm run dev

Runs on:

http://localhost:5173
